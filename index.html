Explanation of statistics

Fitness
the sum of the differences between the evolved and target functions. Lower scores indicate greater fitness.
Size
the number of operators, variables, and constants in an individual. For example, the individual ((3 + 5) * (x / 8)) has a size of 7.
Time
the time taken to run the generation—that is, to generate all individuals within the generation and evaluate each for fitness.
