What is it?

Gee Pee is a genetic programming system that attempts to evolve an expression closely approximating y=sin(x) over the interval [0, 2Ï€], using only the four arithmetical operations (addition, subtraction, multiplication, and division). It was inspired by TinyGP.

Browser support

  • Firefox 3.6 and later: Gee Pee works perfectly.
  • Google Chrome 4.1 and later: Gee Pee works sporadically. If it initializes correctly, it works well; if no graph appears after a few seconds, please close the browser tab and re-open Gee Pee in a new one. (Refreshing the page does not work.)
  • Internet Explorer 8: Gee Pee does not work.
  • Opera 10.5: Gee Pee does not work.

Explanation of terms

Fitness
the sum of the differences between the evolved and target expressions. Lower scores indicate greater fitness.
Individual
an evolved mathematical expression, composed of combinations of the four arithmetical operators, the input variable x, and constant values.
Size
the number of operators, variables, and constants in an individual. For example, the individual ((3 + 5) * (x / 8)) has a size of 7.
Time
the time taken to run the generation—that is, to generate all individuals within the generation and evaluate each for fitness.
